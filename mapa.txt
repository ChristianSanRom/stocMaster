//EXPLICACION DE QUE SE IMPLEMENTO


1. Estructura de Archivos:
Todos los panels principales: Cambiados de .html a .php
Nueva carpeta php/ con subcarpetas por rol:
php/admin/ - Funcionalidades de administrador
php/operario/ - Funcionalidades de operario
php/gerente/ - Funcionalidades de gerente
php/cliente/ - Funcionalidades de cliente
php/config/ - Configuración de conexión

2. Sistema Modal:
Todas las funcionalidades ahora se manejan mediante modales en un solo archivo PHP
No hay navegación entre páginas - todo se carga dinámicamente
Mejor experiencia de usuario sin recargas de página

3. Nuevas Funcionalidades por Rol:
Administrador:
Gestión completa de usuarios, proveedores, productos
Visualización de inventario
Reportes detallados

Operario:
Registrar entradas/salidas de mercancía
Verificar mercancía en almacén
Consultar movimientos

Gerente:
Autorizar/rechazar entradas y salidas
Supervisar inventario con análisis
Reportes gerenciales
Dtalles completos de movimientos

Cliente:
Perfil del cliente
Gestión de pedidos (crear, ver, cancelar)
Catálogo de productos disponibles
Carrito de compras interactivo
Seguimiento de pedidos

4. Tecnologías Implementadas:
SweetAlert2 para notificaciones modernas
AJAX/Fetch para carga dinámica
Validaciones en tiempo real
Transacciones en base de datos
Interfaz responsiva y moderna

5. Estructura de Base de Datos Utilizada:
Todas las funcionalidades están diseñadas para trabajar con tu estructura existente de:
usuarios, clientes, proveedores
productos, mercancia, inventario
entrada_mercancia, salida_mercancia
pedido, item_pedido
reportes

6. Ventajas del Nuevo Sistema:
Una sola página por rol (menos archivos que mantener)
Carga más rápida (solo se carga el contenido necesario)
Mejor experiencia de usuario (sin recargas)
Código más organizado (separación por carpetas)
Mantenimiento más fácil (cambios centralizados)
